# Установка PvPKits Manager

## Шаг 1: Установка Rust

### Windows
1. Скачайте rustup с https://rustup.rs/
2. Запустите установщик
3. Следуйте инструкциям

### Проверка установки
```bash
rustc --version
cargo --version
```

## Шаг 2: Сборка приложения

```bash
cd pvpkits-manager
cargo build --release
```

Или используйте `build.bat` на Windows:
```bash
build.bat
```

## Шаг 3: Запуск

Исполняемый файл находится в:
```
target/release/pvpkits-manager.exe
```

Просто запустите его двойным кликом!

## Первая настройка

1. При первом запуске укажите пути:
   - **Путь к серверу**: где находится ваш Minecraft сервер (например: `C:\servers\test-server`)
   - **Путь к плагину**: где находится исходный код плагина (например: `C:\projects\pvpkits`)

2. Нажмите "Сохранить настройки"

3. Теперь можете:
   - Собрать плагин кнопкой "Собрать и установить"
   - Запустить сервер кнопкой "Запустить"

## Требования

- Windows 10/11
- Java 21+ (для запуска Minecraft сервера)
- Maven (для сборки плагина)
- 4GB+ RAM

## Устранение проблем

### Ошибка "Java not found"
Установите Java 21: https://adoptium.net/

### Ошибка "Maven not found"
Установите Maven: https://maven.apache.org/download.cgi

### Сервер не запускается
Проверьте:
1. Правильность пути к серверу
2. Наличие файла `paper.jar` в папке сервера
3. Достаточно ли RAM

## Обновление

Для обновления просто пересоберите:
```bash
git pull
cargo build --release
```
